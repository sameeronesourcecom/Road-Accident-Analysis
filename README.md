# Road-Accident-Analysis:

**Project Overview:**
This project analyzes road accident data across the UK for the years 2021 and 2022.It includes KPIs, vehicle type breakdowns, urban vs rural distribution, light conditions, and geographic mapping. The goal is to provide stakeholders with insights into accident trends, severity, and contributing factors using Power BI.

**Client Requirements:**
*Primary KPIs:Total Casualties and Total Accidents for Current Year with YoY growth
*Casualties by Accident Severity (Fatal, Serious, Slight) for Current Year with YoY growth
*Secondary KPIs:Casualties by Vehicle Type for Current Year
*Trend Analysis: Monthly comparison of casualties for Current Year vs Previous Year

**Breakdown Analysis:**
Casualties by Road Type
Casualties by Area/Location (Urban vs Rural)
Casualties by Day vs Night
Total Casualties and Accidents by Location (map view)

**Steps in Project:**
01. Requirement Gathering
02. Identifying Stakeholders
03. Raw Data Overview
04. Connecting Data with Power BI
05. Data Cleaning
06. Data Processing
07. Data Modelling
08. Visualization and Chart Design
09. Dashboard Building and Insights
